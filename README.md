# Cognifyz Technologies – Java Programming Internship

This repository contains the Java programming tasks completed as part of the **Cognifyz Technologies Internship Program**. Each task focuses on core Java concepts such as control flow, string manipulation, OOP, and basic algorithm design.

## 📋 Tasks Overview

| Task | Description | Level |
|------|-------------|-------|
| **Task 1 – Temperature Converter** | Converts temperature values between Celsius, Fahrenheit, and Kelvin. | Level 1 |
| **Task 1 – Tic-Tac-Toe (Two Players)** | A console-based two-player Tic-Tac-Toe game with turn-based input and win/draw detection. | Level 1 |
| **Task 2 – Palindrome Checker** | Checks whether a given string or number is a palindrome. | Level 2 |
| **Task 2 – Password Strength Checker** | Evaluates password strength based on length, character variety, and common security rules. | Level 2 |
| **Task 3 – Student Grade Calculator** | Calculates grades/percentage based on marks entered by the user. | Level 3 |
| **Task 4 – Random Password Generator** | Generates secure random passwords based on user-defined length and character sets. | Level 4 |

## 🛠️ Tech Stack

- **Language:** Java
- **Execution:** Console-based (CLI) programs
- **Concepts used:** Conditionals & loops, string handling, arrays, basic OOP, input validation

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/cognifyz.git
   cd cognifyz
   ```
2. Compile any task file:
   ```bash
   javac "Task Name.java"
   ```
3. Run it:
   ```bash
   java "Task Name"
   ```
